## 📌 项目

### 🎓 家教服务平台

🔹 一个面向学生与家教老师的 **在线匹配与管理平台**  
🔹 提供从发布需求到匹配教师、沟通与管理的一体化服务  

💡 技术方向：

- Spring Boot + Vue 前后端分离架构
- JWT 用户认证与权限控制（学生 / 教师）
- Redis 缓存热点数据（教师列表 / 推荐）
- MySQL 存储用户、订单与匹配关系
- RabbitMQ（可选）处理异步通知 / 任务

🔹 **核心功能：**

- 👨‍🏫 教师入驻与资质管理
- 🎯 学生发布家教需求
- 🔍 智能匹配 / 条件筛选
- 💬 在线沟通 / 留言机制
- 📊 订单与教学记录管理

🔹 **系统特点：**

- 分角色权限控制（学生 / 教师 / 管理员）
- 高并发场景下的数据缓存优化
- 可扩展的业务模型（支持未来增加 AI 推荐）
